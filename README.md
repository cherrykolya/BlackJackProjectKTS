 # BlackJackBot VK
 Этот проект выполнен в качестве выпускного в backend-школе от KTS

 ## Идея
 - Целью данного проекта была реализация игрового бота в VK.
 - Бот позволяет играть в БлэкДжек при помощи клавиатуры.
